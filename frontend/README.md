# Frontend (React + TypeScript + D3)

## Run
```bash
cd frontend
npm install
npm run dev
# opens http://localhost:5173 (proxies /api to Flask at 5001)
```

## Features
- T1: Two force-directed graphs (citation, collaboration) with draggable nodes and tooltips
- T2: Coordinated timeline and histogram (click a year to update histogram)
- T3: Refined network using hierarchical edge bundling and tuned layout
- Titles/legends for readability; scalability paragraph embedded in UI
